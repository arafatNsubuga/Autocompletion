** 1. Describe how your firstIndexOf() method in RangeBinarySearch.java
**    finds the first index of a key that equals the search key.
******************************************************************************/

[The firstIndexOf() method uses a modified binary search algorithm to find the first occurrence of a key in a sorted array. 
It maintains a result variable initialized to -1 and performs standard binary search with a key modification: when an equal element is found (comparator.compare(terms[mid], key) == 0), instead of returning immediately, 
it stores the current index in result and continues searching in the left half of the array by setting right = mid - 1. This ensures that if there are multiple matching elements, the search continues leftward until finding the first occurrence. 
The method returns the stored result, which will be the leftmost (first) index of all matching elements, or -1 if no match is found. ]
